Network filter

Filter network by node parameter:

1. You can set top n nodes by parameter, where 1 ≤ n ≤ 100.

2. The node parameters are as follows:

  • Representativeness: Description
  • Raw occurrence: Description
  • Betweenness centrality: Description
  • Closeness centrality: Description
  • Strength: Description
  • W-score: Description
  • K-node: Description
  • K-node rank: Description
  • Community membership: Description

        


Filter network by node list:

1. Provide n nodes by name, where 1 ≤ n ≤ 100.

2. Resultant network shall contain no more than the nodes specified by user.

3. If the resultant network contains more than 100 nodes, you will see a message: "More than 100 nodes in resultant network; please enter fewer number of nodes to visualize!".

4. Follow the following rules while providing node list:

  • Individual gene/ mesh terms: Node names should not be enclosed within inverted commas.
  • White spaces: If node name contans a space, just write is at is.
  • Separators: Node names should be separated by a single comma, with no additional spaces in between.

Filter network by expanded node list:

1. Provide n nodes by name, where 1 ≤ n ≤ 100.

2. Resultant network shall contain no more than the nodes specified by user, and their immediate neighbors.

3. If the resultant network contains more than 100 nodes, you will see a message: "More than 100 nodes in resultant network; please enter fewer number of nodes to visualize!".

4. Follow the following rules while providing node list:

  • Individual gene/ mesh terms: Node names should not be enclosed within inverted commas.
  • White spaces: If node name contans a space, just write is at is.
  • Separators: Node names should be separated by a single comma, with no additional spaces in between.
